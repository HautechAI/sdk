# Changelog

## Unreleased

## 0.1.0 - 2026-02-24

### Added

- Added the `onRequestError` hook to `createSDK`/`createDirectorySDK` for centralized HTTP retry handling.
- Exposed `ws.disconnect()` on the websocket client to support explicit shutdown.
- Added Seedream v5 Lite text-to-image (`seedream.5_lite_t2i.v1`) and edit (`seedream.5_lite_edit.v1`) operations to the SDK, including autogenerated clients, manual mappings, tests, and documentation updates noting the $0.04375/run pricing.

### Usage

```ts
const sdk = createSDK({
    authToken: getCurrentToken,
    onRequestError: async ({ error }, { attempt }) => {
        if (error.status === 401 && attempt === 1) {
            await refreshToken();
            return { retry: true, invalidateToken: true, backoffMs: 200 };
        }

        return { retry: false };
    },
});
```
